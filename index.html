SVG path

This is a preview of our SVG technology. The path/stroke of the image is defined mathimatically. This negates the issue of pixelation when upscaling, it is very useful when it comes to various positions and sizes, as it will always be the same image. Other factors of it can be edited very easily too.

PNG image

This simple image format is very cheap to produce and create, however there is a downside, the image is a fixed pixelated resolution which can cause it to become pixelated and blocky when resizing. It is also limited in color options as each one has to be saved individually, where as SVG and Bezier can be altered very easily.
